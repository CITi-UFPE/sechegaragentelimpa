# sechegaragentelimpa
O #SeChegarAGenteLimpa é uma plataforma digital para suportar a causa, mapeando os pontos de incidência do óleo e conectando pessoas, recursos e empresas. Se chegar a gente limpa é um movimento de propósito compartilhado com foco em pessoas e empresas que querem fazer a diferença de maneira voluntária e colaborativa!
